# a-plus-skill repository

이 저장소의 실제 코드 프로젝트 루트는 `a-plus-skill/` 디렉터리입니다.

## 빠른 시작
```bash
cd a-plus-skill
npm ci --include=dev
npm run test
npm run build
npm run dev
```

> `npm ci --include=dev`를 권장합니다. `--omit=dev`로 설치하면 `dev/test/build` 스크립트가 실패할 수 있습니다.
